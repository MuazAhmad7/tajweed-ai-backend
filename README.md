# TajweedAI Backend

Simple Express.js backend server for the TajweedAI application.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. The server will run on `http://localhost:5000`

## Available Routes

- `GET /` - Welcome message
- `GET /health` - Health check endpoint
